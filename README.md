# Jira Ops Lab – IAM Access Request Ops

This repo documents a Jira configuration lab demonstrating:
- Issue hierarchy (Initiative/Epic/Story/Task/Sub-task), Bugs, Service Requests
- Scrum vs Kanban boards (WIP limits, swimlanes, quick filters)
- Workflows (approvals/gating, Definition of Done, SLA states concept)
- JQL library, dashboards, reporting
- Integration patterns (GitHub, Slack/Teams, ServiceNow handoffs – conceptual)
